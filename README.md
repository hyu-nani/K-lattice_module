# K-lattice_module
한국식 격자 모듈 전용 STM 펌웨어 
- USB uart 를 통한 문자 지정 출력기능
- CasCade 유사 방식을 통해 다중 디바이스 연결 구성 가능
- 특수 문자 출력 및 RGB 출력
- 점진적 밝기 변화방식으로 부드러운 밝기 변화
- 개별 모듈 당 temp, humi 및 bright 센서를 통해 모듈마다 개별 기능 구사
- CDS 센서를 통해 주변 밝기에 반응하여 밝기 전환
- 날짜, 시간, 문자열, 온도, 습도 출력기능
